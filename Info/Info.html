HTML

HTML stands for HyperText Markup Language. It is the standard language for creating web pages and web applications. With this the structure of a webpage is defined using elements and tags.

CSS

CSS stands for Cascading Style Sheets. It is used to style and layout web pages, including the design, colors, and fonts. This the Website to have a format and style.

JavaScript

JavaScript is a programming language that allows you to create dynamic and interactive content on web pages. It is used to enhance user experience by adding features such as animations, form validation, and interactive maps.

Visual Studio Code

Visual Studio Code is used for developing computer programs, websites, web apps, web services, and mobile apps. It is a source-code editor made by Microsoft for Windows, Linux and macOS.
